# Demon Slayer RPG

## Descrição
Demon Slayer RPG é um jogo de batalhas 1x1 entre personagens **Slayers** e **Oni**, com sistema de ataques, esquivas, efeitos de veneno e progressão de experiência (XP).

## Como Jogar
1. Faça login com seu nome de jogador (sem senha).
2. Escolha seu personagem.
3. Batalhe contra outro jogador.
4. Vença batalhas para ganhar XP e subir de nível.

## Personagens
O jogo possui personagens únicos, cada um com:
- Vida
- Esquiva
- Ataques especiais (alguns com efeito de veneno)
- Tipo de respiração (para Slayers)
- Habilidades de regeneração (para Onis)

## Sistema de XP e Níveis
- Cada vitória dá XP ao vencedor.
- Quanto mais você joga, mais difícil fica subir de nível.
- O progresso de nível aparece ao iniciar a batalha.
